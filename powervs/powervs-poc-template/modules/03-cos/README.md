# Cloud Object Storage Module

This module creates COS instance and buckets with encryption.

## Module Information

- **Module Source**: `terraform-ibm-modules/cos/ibm`
- **Version**: `10.5.0`
- **Purpose**: Object storage for data and backups

## Resources Created

- COS Instance
- COS Buckets
- Encryption configuration

## Dependencies

None - Independent service

## Outputs

- `cos_instance_id`: COS instance identifier
- `cos_instance_crn`: COS instance CRN
- `bucket_id`: Bucket identifier
